DMC TV Server Live!

-

maxresdefault 12/10/2024

After a long journey of trial and error with many hiccups and failures, the official DMC TV server is now live! Our server is hosted in San Jose 24/7 and mostly anyone in North America should have playable ping. You can connect to the server by clicking the link on the left side of our website (if DMC is installed), or by entering play.dmctv.net into your server browser.

The goal of this server is to provide a vanilla experience supplemented with bots (it is DMC after all). The thing that differentiates our server from others is our bot plugin. We are running the most advanced bot plugin available for GoldSrc. Other popular DMC servers use a plugin called HPB-Bot which relies on pre generated navigation files for maps. Of course almost any custom map won't have these navigation files generated and the bots thus won't play those maps well. Even on maps with these navigation files HPB-Bot is quite limited, the bots do not jump, or play very well.

Our server is running a custom build of Parabot compiled by me. I have contributed some code changes and bug fixes to this project, but overall Parabot is amazing. This plugin generates navigation files from the way players move and play; the bots learn! These bots are far more aggressive and smart, and sometimes they really do feel like real humans. Unfortunately, Parabot is a deprecated project and was riddled with bugs. This is where our custom build comes in, it took months to fix all the bugs I had found. But now I can confidently say our version of the plugin is stable, and ready to go public on the server!

Other notable additions include a custom compiled version of AMX Mod X's map voting plugin. This version modified by me will display properly on the Half-Life 25th Anniversary update.

TLDR: When Valve published this update, they modified the code surrounding UI scaling, so many old plugins in DMC (and other GoldSrc games) no longer display correctly and are sometimes even unreadable. This will not be an issue on our server (even if you are playing on the legacy branch of the game the map vote UI will still be readable). The other change is that when no votes are cast, a random map will be selected, instead of falling back to the next map in the map cycle.

The current map cycle for the server is:

  • dmc_dm2

Lastly, I have other plugins in the works! I am currently working on a plugin that will upload your in game statistics to this website! This might still be a ways away, but it is being worked on. Always check back on our website for any updates. Enjoy the server!
